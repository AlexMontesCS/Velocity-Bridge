# Contributing to Velocity Bridge

Thanks for considering contributing!

## How to Help

**Report bugs** — Open an issue describing what happened and how to reproduce it.

**Suggest features** — Open an issue with your idea.

**Submit code** — Fork, make changes, submit a PR.

## Development Setup

```bash
git clone https://github.com/Trex099/Velocity-Bridge.git
cd velocity
pip install -r requirements.txt
python3 -m uvicorn main:app --reload --port 8080
```

## Code Style

- Python 3.10+
- No external linter enforced, just keep it readable
- Test your changes before submitting

## Questions?

Open an issue.
